# RoadGuard-Pro2
نظام ذكي للتنبيه المبكر وتتبع المسارات (Real-time GPS Tracking) مصمم خصيصاً لتحسين السلامة المرورية على طرق محافظة الشرقية (بلبيس - العاشر) و (فاقوس - الحسينية). يعتمد النظام على تقنيات التحليل المكاني (Spatial Analysis) لتنبيه السائقين عند الاقتراب من النقاط السوداء (Black Spots) والمناطق عالية الخطورة
